# Ika Studio ComfyUI Worker

Custom RunPod serverless worker with animagine-xl for anime image generation.

## Models Included
- `animagine-xl.safetensors` - Animagine XL 3.1 (anime-focused SDXL finetune)

## Usage
Deploy as RunPod serverless endpoint using:
```
ghcr.io/illuminfti/ika-comfyui-worker:latest
```

## API
Standard ComfyUI workflow JSON via RunPod serverless API.
